# miniRT
42 project to build a ray tracer in C. This program renders an image based on a given scene.

## Scene elements

Resolution `R`
Ambiant light `A`
Camera `c`
Light `l`
Sphere `sp`
Plane `pl`
Square `sq`
Cylinder `cy`
Triangle `tr`

## How to run

Use `make` to create `miniRT` executable.

Now run with :

```
./miniRT scene.rt [--save]
```

`--save` option saves the result in a bmp file.
